## 🌈 色彩系統升級

### 新的色彩變數
我們重新設計了整個色彩系統，包含：

- **主色調**: 保持原有的綠色主題，但增加了更多變化
- **功能色彩**: 新增成功、警告、錯誤、資訊等狀態色彩
- **漸變效果**: 添加了美麗的漸變背景
- **亮色主題**: 完善的明亮主題支援

```css
:root {
  /* 主要品牌色 */
  --pri: #6ee7b7;
  --pri-hover: #34d399;
  --pri-light: rgba(110, 231, 183, 0.1);
  
  /* 功能色彩 */
  --success: #10b981;
  --warning: #f59e0b;
  --error: #ef4444;
  --info: #3b82f6;
}
```

## 🚀 快速開始（3 步驟）

```powershell
# 1. 安裝依賴
bundle install

# 2. 啟動開發伺服器
.\start-dev.ps1

# 3. 瀏覽器打開 http://localhost:4000
```

## 📝 寫文章

```powershell
# 快速建立文章
.\tools\quick-post.ps1 -Title "文章標題" -Category "分類"

# 統一管理工具
.\tools\master-tool.ps1
```
